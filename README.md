# Deep-Learning
This repository contains a set of deep learning experiments covering image classification, natural language processing, and regression tasks. It includes implementations of MLPs, CNNs, RNNs, LSTMs, autoencoders, and transfer learning using datasets like MNIST, IMDB, Reuters, and Boston Housing for hands-on learning.


### Core Experiments

### Core Experiments

**Experiment 1: Multilayer Perceptron for MNIST**
This experiment focuses on building a basic neural network to classify handwritten digits from the MNIST dataset using fully connected layers.

**Experiment 2: One-Hot Encoding**
This experiment demonstrates how text data such as words or characters can be converted into numerical format using one-hot encoding.

**Experiment 3: Data Augmentation**
This experiment applies techniques like rotation, flipping, and scaling to images to increase dataset size and improve model performance.

**Experiment 4: Word Embeddings for IMDB**
This experiment converts movie review text into dense vector representations to capture semantic meaning using word embeddings.

**Experiment 5: IMDB Movie Review Classification (Binary)**
This experiment builds a neural network to classify movie reviews as positive or negative based on sentiment.

**Experiment 6: Reuters News Classification (Multiclass)**
This experiment designs a neural network to categorize news articles into multiple topics using the Reuters dataset.

**Experiment 7: Boston Housing Price Prediction**
This experiment uses a neural network to predict house prices based on various input features.

**Experiment 8: CNN for MNIST**
This experiment implements a convolutional neural network to improve handwritten digit classification accuracy using image features.

**Experiment 9: CNN for Dogs vs Cats**
This experiment classifies images of dogs and cats and studies how batch normalization and dropout affect model performance.

**Experiment 10: VGG16 Pre-trained Model**
This experiment uses a pre-trained VGG16 model to perform image classification through transfer learning.

**Experiment 11: RNN for IMDB Reviews**
This experiment applies a recurrent neural network to analyze sequential text data for movie review sentiment classification.

**Experiment 12: Denoising Autoencoder on MNIST**
This experiment trains an autoencoder to remove noise from handwritten digit images and reconstruct clean images.
